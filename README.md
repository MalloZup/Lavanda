# Lavanda
Lavanda is a ruby library, testing utilities builded around twopence.

The library is published in rubygem.


## How install and use Lavanda:

`gem install Lavanda.`

## How to use Lavanda


In  a `hello.rb` file

`require twopence`
`require lavanda`
